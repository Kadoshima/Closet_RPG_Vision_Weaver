# Closet RPG: VisionWeaver 利用方法

このアプリケーションは、Google Gemini APIを活用し、RPGの装備生成のような没入感のある体験を通じて、理想のファッションアイテムをデザイン・探索できるツールです。

## はじめに

ブラウザでアプリケーションを開くと、ダークファンタジー風のUIが表示されます。
画面の指示に従ってカードを選択していくことで、AIが画像を生成します。

## 基本的な使い方 (クエストの流れ)

### 1. ターゲットの選択 (Identify Target)
最初に、誰向けのアイテムかを選択します。
*   **Men's** / **Women's** / **Unisex**

### 2. カテゴリの選択 (Select Class)
作成したいアイテムの大分類を選びます。
*   **Bag** (鞄) / **Footwear** (靴) / **Outerwear** (アウター) / **Tops** (トップス)

### 3. サブカテゴリの選択 (Refine Class)
より具体的な種類を選びます。
*   例: Bagを選択した場合 → Backpack, Tote, Messengerなど

### 4. スタイルの選択 (Choose Archetype)
デザインの方向性（アーキタイプ）を選びます。
*   **Minimalist**: シンプルで洗練されたデザイン (Stealth +5)
*   **Rugged**: 武骨で耐久性のあるデザイン (Toughness +10)
*   **Luxury**: 高級感のあるデザイン (Charisma +20) など

### 5. ムードの選択 (Infuse Essence)
アイテムに纏わせる雰囲気を選びます。これを選択すると**生成(Forging)**が始まります。
*   **Noir**: 黒基調、ミステリアス
*   **Earth**: 自然色、オーガニック
*   **Cyber**: ネオン、近未来的
*   **Ethereal**: パステル、軽やか

### 6. 生成結果の確認 (Forge Complete)
AIが4つのバリエーション画像を生成します。
*   気に入った画像のカードをクリックして詳細画面へ進みます。
*   各カードには、生成されたアイテム名が表示されます。
*   クローゼット連携を行っている場合、相性スコアも表示されます。

## 詳細機能

### Item Detail (詳細画面)
選択したアイテムの以下の情報が生成されます。
*   **高解像度画像**: 選択したアイテムの拡大画像。
*   **Item Lore**: アイテムの物語、属性(Element)、フレーバーテキスト。
*   **Item Statistics**: 耐久性(Durability)、収納力(Storage)、カリスマ(Charisma)などのRPG風ステータス。

### My Closet Sync (クローゼット連携)
自分の持っている服との相性を診断できます。

1.  画面右上の **「👕」アイコン** をクリックしてサイドバーを開きます。
2.  **「+ Sync Equipment」** エリアをクリックし、自分の服の写真をアップロードします。
3.  AIが画像を分析し、色・スタイル・素材を抽出します。
4.  アイテム生成時に、アップロードした服との **相性スコア (Compatibility Score)** が計算され、マッチ度が可視化されます。
    *   詳細画面では「Closet Sync: 85%」のように表示されます。

### Vibe Refinement (音声による微調整)
詳細画面の左側にあるマイクボタンを使って、感覚的にデザインを修正できます。

1.  **マイクボタン** をクリックして録音を開始します。
2.  修正したい内容を話しかけます。具体的な指示だけでなく、擬音（オノマトペ）も理解します。
    *   「もっと **クタッと** させて」
    *   「**ドサッ** とした重厚感がほしい」
    *   「金具をゴールドにして」
3.  もう一度ボタンを押して録音を停止すると、AIが音声を解釈し、アイテムを「再鍛造 (Reforge)」して画像を更新します。

## その他
*   **Reset Quest**: 画面上部のプログレスバー右端にあるボタンで、現在の進行状況をリセットし、最初からやり直すことができます。

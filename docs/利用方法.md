# VisionWeaver Studio 利用方法

VisionWeaver Studioは、Google Gemini APIを活用した次世代のAIファッションデザインスタジオです。
好みのスタイルを選択するだけで、あなただけのオリジナルファッションアイテムをデザインしたり、手持ちの服との相性を診断したり、画像からアイテムを検索したりすることができます。

## はじめに

ブラウザでアプリケーションを開くと、モダンで洗練されたデザインスタジオのインターフェースが表示されます。
ステップバイステップで直感的に操作できます。

## デザインと生成の流れ (Design Flow)

### 1. ターゲットの選択 (Design Target)
誰向けのアイテムをデザインするか選びます。
*   **Men** / **Women** / **Unisex**

### 2. カテゴリの選択 (Category & Specifics)
アイテムの種類を選びます。
*   **Tops**, **Outerwear**, **Bottoms**, **Shoes**
*   その後、具体的なサブカテゴリ（例: T-Shirt, Coat, Denimなど）を選択します。

### 3. スタイルの選択 (Aesthetic)
デザインの方向性を選びます。
*   **Minimal**: ユニクロUのようなシンプルで洗練されたスタイル
*   **Street**: 大胆でトレンド感のあるストリートスタイル
*   **Classic**: タイムレスで上品なスタイル
*   **Casual**: 日常的に使いやすいリラックススタイル

### 4. シチュエーションの選択 (Context)
着用シーンを選びます。これを選択するとAIによる**デザイン生成**が始まります。
*   **City Life**: 都会的でスマート
*   **Vacation**: リゾートや休暇向け
*   **Office**: ビジネス、フォーマル
*   **Weekend**: 週末のリラックスタイム

### 5. コレクションの確認 (Collection Ready)
AIが4つのバリエーションを提案します。
*   気に入ったアイテムをクリックして詳細を確認しましょう。
*   クローゼットに服を登録している場合、**Match Score (相性度)** も表示されます。

## 主要機能

### Product Detail (商品詳細)
選択したアイテムの以下の情報が生成されます。
*   **Product Info**: 商品名、魅力的な説明文、素材情報。
*   **Styling Tips**: プロのスタイリストによる着こなしアドバイス。
*   **Specs**: Versatility（着回し力）、Comfort（快適さ）、Trend（トレンド感）の指標。
*   **Find Similar**: 生成されたアイテムに近い商品をGoogleショッピング（Uniqlo, H&M, Zaraなど）で検索するリンクが表示されます。

### Visual Search (画像検索)
雑誌やSNSで見かけたアイテム、あるいは手持ちのアイテムの写真をアップロードして、似た商品を検索できます。

1.  ヘッダーの **カメラアイコン** をクリックして検索画面を開きます。
2.  探したいアイテム（コートやバッグなど）の画像をアップロードします。
3.  AIが画像を分析し、特徴を解説するとともに、オンラインで購入可能な類似商品やリンクをリストアップします。

### My Wardrobe (クローゼット連携)
自分の持っている服との相性をチェックできます。

1.  画面右上の **「Wardrobe」ボタン** をクリックしてサイドバーを開きます。
2.  **「+ Add Item」** エリアをクリックし、自分の服の写真をアップロードします。
3.  AIが服の色やスタイルを解析し、デジタルクローゼットに保存します。
4.  新しいアイテムを生成する際、クローゼットのアイテムとの **相性スコア** が自動的に計算されます。

### AI Designer Assistant (音声修正)
詳細画面にあるマイクボタンを使って、デザインを微調整できます。

1.  **マイクボタン** をクリックして、「もっと明るい色にして」「丈を短くして」「よりフォーマルに」などと話しかけます。
2.  AIが指示を理解し、デザインを即座に修正（Re-design）して新しい画像を生成します。

## その他
*   **Start Over**: ヘッダーのリンクをクリックすると、現在の進行状況をリセットし、最初からやり直すことができます。
